Cloudflareキャッシュ削除

  • URLはhttps://から入力します。
  • smcworld.com と smc.com.cn は混在出来ません。
  • 改行区切りで複数入力可能です(最大100個)
  • リクエスト制限もあるので連打しないようにしてください
   ・・・個別にファイルを指定します。
クエリ(?～)、ハッシュ(#～)も入力可能です。
   ・・・ディレクトリ、ファイル配下をすべて削除します。
クエリ(?～)、ハッシュ(#～)は入力出来ません。
例) https://www.smcworld.com/abc.html
→ https://www.smcworld.com/abc.html    https://www.smcworld.com/abc.html?q1=aaa&q2=bbb
   ・・・下記ホストを指定します。
www.smcworld.com    ca01.smcworld.com    www.smc.com.cn
